Macеҝ«жҚ·й”®

Yu's Blog
2020-07-04

й”®зӣҳз¬ҰеҸ·иҜҙжҳҺ

вҢҳ == Command

вҮ§ == Shift

вҮӘ == Caps Lock

вҢҘ == Option

вҢғ == Control

вҶ© == Return/Enter

вҢ« == Delete

вҢҰ == еҗ‘еүҚеҲ йҷӨй”®пјҲFn+Deleteпјү

вҶ‘ == дёҠз®ӯеӨҙ

вҶ“ == дёӢз®ӯеӨҙ

вҶҗ == е·Ұз®ӯеӨҙ

вҶ’ == еҸіз®ӯеӨҙ

вҮһ == Page UpпјҲFn+вҶ‘пјү

вҮҹ == Page DownпјҲFn+вҶ“пјү

Home == Fn + вҶҗ

End == Fn + вҶ’

вҮҘ == еҸіеҲ¶иЎЁз¬ҰпјҲTabй”®пјү

вҮӨ == е·ҰеҲ¶иЎЁз¬ҰпјҲShift+Tabпјү

вҺӢ == Escape (Esc)

вҸҸ == з”өжәҗејҖе…ій”®

зЁӢеәҸжҺ§еҲ¶

вҢҳ+QпјҡйҖҖеҮәзЁӢеәҸ

вҢҳ+Wпјҡе…ій—ӯзӘ—еҸЈ

вҢҳ+MпјҡжңҖе°ҸеҢ–

вҢҳ+Hпјҡйҡҗи—ҸеҪ“еүҚзӘ—еҸЈ

вҢҳ+вҢҘ+Hпјҡйҡҗи—Ҹе…¶д»–жүҖжңүзӘ—еҸЈ

вҢҳ+TabпјҡеҲҮжҚўеә”з”Ё

Finder еҝ«жҚ·й”®

вҢҳ+Nпјҡжү“ејҖж–°зӘ—еҸЈ

вҢҳ+EпјҡжҺЁеҮәжүҖйҖүзЈҒзӣҳжҲ–е®—еҚ·

вҮ§+вҢҳ+Oпјҡжү“ејҖж–ҮзЁҝж–Үд»¶еӨ№

вҮ§+вҢҳ+Dпјҡжү“ејҖжЎҢйқўж–Үд»¶еӨ№

вҢҳ+Deleteпјҡе°Ҷж–Үд»¶з§»иҮіеәҹзәёзҜ“

жөҸи§ҲеҷЁеҝ«жҚ·й”®

вҢҳ+Tпјҡж–°е»әж ҮзӯҫйЎө

вҮ§+вҢҳ+TпјҡжҒўеӨҚе…ій—ӯзҡ„ж ҮзӯҫйЎөпјҲChromeпјү

вҢҳ+Lпјҡеҝ«йҖҹиҫ“е…ҘзҪ‘еқҖ

вҢҳ+RпјҡеҲ·ж–°йЎөйқў

вҢҳ++/вҖ”пјҡж”ҫеӨ§/зј©е°ҸйЎөйқў

вҢҳ+0пјҡжҒўеӨҚй»ҳи®ӨеӨ§е°Ҹ

вҮ§+з©әж јй”®/з©әж јй”®пјҡж»ҡеҠЁйЎөйқў
