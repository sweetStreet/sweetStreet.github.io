bisect

leetcode里要import bisect

data = [2,4]
bisect.insort(data, 3)

data = [2,3,4]
插入树并排序

insort和bisect的区别在于bisect有返回值
1、bisect，返回插入的插入值9的位置，但是并未插入

>>>data = [4,5,6,7,8]
>>>bisect.bisect(data,9)
5

2、bisect_left，左边插入值，并返回插入的index

>>>data = [4,5,6,7,8]
>>>bisect.bisect_left(data,5)
1

3、bisect_right，右边插入值，并返回插入的index

>>>data = [4,5,6,7,8]
>>>bisect.bisect_right(data,5)
2

4、insort，插入值，但不会改变原来的顺序

>>>data = [4,5,6,7,8]
>>>bisect.insort(data,9)
>>>data
[4,5,6,7,8,9]

5、insort_left,左边插入值，不返回

>>>data = [4,5,6,7,8]
>>>bisect.insort_left(data,5)
>>>data
[4,5,5,6,7,8]

6、insort_right ,右边插入值，不返回。

>>>data = [4,5,6,7,8]
>>>bisect.insort_right(data,5)
>>>data
[4,5,5,6,7,8]
