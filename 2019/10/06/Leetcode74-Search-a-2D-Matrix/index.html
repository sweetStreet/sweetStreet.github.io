[Leetcode74]Search_a_2D_Matrix

Yu' Blog
2019-10-06
algorithm, binary search, leetcode

Write an efficient algorithm that searches for a value in an m x n matrix. This matrix has the following properties:

Integers in each row are sorted from left to right.
The first integer of each row is greater than the last integer of the previous row.

Example 1:

Input:
matrix = [
  [1,   3,  5,  7],
  [10, 11, 16, 20],
  [23, 30, 34, 50]
]
target = 3
Output: true

Example 2:

Input:
matrix = [
  [1,   3,  5,  7],
  [10, 11, 16, 20],
  [23, 30, 34, 50]
]
target = 13
Output: false

solution 1
先对第一列做二分查找，再对行做二分查找
二分查找模版

  1. 查找和目标完全相等的数

  2. 查找第一个大于等于目标值的数，可变形为查找最后一个小于目标值的数。比如在数组 [2, 4, 5, 6, 9] 中查找数字3，就会返回数字4的位置；在数组 [0, 1, 1, 1, 1] 中查找数字1，就会返回第一个数字1的位置。

变形：查找最后一个小于目标值的数，怎么变呢。我们已经找到了第一个大于等于目标值的数，那么再往前退一位，返回 right - 1，就是最后一个小于目标值的数。

  1. 查找第一个大于目标值的数，可变形为查找最后一个不大于目标值的数

注意处理边界情况

solution 2

则一维数组第i个数对应的是二维数组[i/n][i%n]
