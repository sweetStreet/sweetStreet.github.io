algorithm, dp

take climbing stairs as an example

  1. top down, no memorization

if there are six steps,

           6
          / \
         4   5
        /\   /\
 /\    2  3 4  6
 ||   /\ /\ 
     0 1 1 2...
     ...
from 0 to 2, we return 1

base case: f(0) = 1, f(-x) = 0
we continuously ask lower layer, can you give me a solution.
the time complexity would be (2^n)
However, there are lots of dupilcation in the tree, in fact, we can cut the subtree of 5, 3 by memorization

           6
          / \
         4   5
        /\   
       2  3 
  /   /\
     0  1 ...
  1. bottom up
    base case: f(0) = 1 f(1) = 1
