[Leetcode955]Delete_Columns_to_Make_Sorted_II

Yu' Blog
2019-10-25
algorithm, greedy, leetcode

We are given an array A of N lowercase letter strings, all of the same length.

Now, we may choose any set of deletion indices, and for each string, we delete all the characters in those indices.

For example, if we have an array A = [“abcdef”,”uvwxyz”] and deletion indices {0, 2, 3}, then the final array after deletions is [“bef”,”vyz”].

Suppose we chose a set of deletion indices D such that after deletions, the final array has its elements in lexicographic order (A[0] <= A[1] <= A[2] … <= A[A.length - 1]).

Return the minimum possible value of D.length.

Example 1:

Input: ["ca","bb","ac"]
Output: 1
Explanation: 
After deleting the first column, A = ["a", "b", "c"].
Now A is in lexicographic order (ie. A[0] <= 0 1 3 a[1] <="A[2])." we require at least deletion since initially a was not in lexicographic order, so the answer is 1. example 2: input: ["xc","yb","za"] output: explanation: already don't need to delete anything. note that rows of are necessarily order: ie. it true (a[0][0] 3: ["zyx","wvu","tsr"] have every column. pre> 


Note:


1 <= A.length <= 100
1 <= A[i].length <= 100
