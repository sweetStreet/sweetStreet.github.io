[Leetcode254]Factor_Combinations

Yu' Blog
2019-10-11
algorithm, leetcode

Description

Numbers can be regarded as product of its factors. For example,

8 = 2 x 2 x 2;
= 2 x 4.
Write a function that takes an integer n and return all possible combinations of its factors.

Note:

You may assume that n is always positive.
Factors should be greater than 1 and less than n.

Example 1:

Input: 1
Output: []

Example 2:

Input: 37
Output:[]

Example 3:

Input: 12
Output:
[
  [2, 6],
  [2, 2, 3],
  [3, 4]
]

solution

注意list.size()>=2 这个条件，说明了单纯数本身是不能被算进来的

  1. backtrace方法中向res数组中添加数组需要new ArrayList<>(list)，否则list是局部变量，会随着方法的结束而变为空
  2. if(n%i==0)这个判断条件要加，不然会超时
  3. 剪枝，如果product*i已经比目标数n大就没有必要继续了，这也是为什么一些combination时，如果是从List中取数，List要排序的原因
