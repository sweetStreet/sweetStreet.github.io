[Leetcode753]Cracking_the_Safe

Yu' Blog
2019-10-23
algorithm, greedy, leetcode

There is a box protected by a password. The password is a sequence of n digits where each digit can be one of the first k digits 0, 1, …, k-1.

While entering a password, the last n digits entered will automatically be matched against the correct password.

For example, assuming the correct password is “345”, if you type “012345”, the box will open because the correct password matches the suffix of the entered password.

Return any password of minimum length that is guaranteed to open the box at some point of entering it.

Example 1:

Input: n = 1, k = 2
Output: "01"
Note: "10" will be accepted too.
Example 2:

Input: n = 2, k = 2
Output: "00110"
Note: "01100", "10011", "11001" will be accepted too.

Note:

n will be in the range [1, 4].
k will be in the range [1, 10].
k^n will be at most 4096.

solution
题目意思就比较难懂，是说密码共有n位，每一位可以有k个数字，所以密码总数就是n^k个，但是因为试密码是以流的方式进行，也就是说密码之间是可以重叠的，所以说当n=2,k=2时，有00 (00110), 01 (00110), 11 (00110), 10 (00110)这四种方式。进一步我们发现，当前后两个密码共享n-1位时，累加出来的密码是最短的。因此采用贪心算法，每次从由n位0组成的密码开始，取现有的n-1位，往后面补一位，使得当前密码串最后n位是一个没有遇到过的新密码。

内层for循环是内层如果改成

class Solution {
public String crackSafe(int n, int k) {
//k^n
StringBuilder sb = new StringBuilder();
Set visited = new HashSet();
for(int i = 0;i<n;i++){
sb.append(‘0’);
visited.add(sb.toString());
}

    int index = (int)(n + Math.pow(k,n)-1);

    while(index>0){            
        for(int i = k-1; i>=0; i--){
            sb.append(i);
            String target = sb.substring(sb.length()-n,sb.length());
            if(!visited.contains(target)){
                visited.add(target);
                break;
            }else{
                sb.deleteCharAt(sb.length()-1);
            }
        }
        index--;
    }

    return sb.toString();
}

}
