[Leetcode206]Reverse Linked List

Yu' Blog
2019-10-09
algorithm, leetcode, reverse list

Description

Reverse a singly linked list.

Example:

Input: 1->2->3->4->5->NULL
Output: 5->4->3->2->1->NULL

Follow up:

A linked list can be reversed either iteratively or recursively. Could you implement both?

Solution 1:

设置一个dummy node，因为刚刚从26题过来，所以很自然地想到了这个方法，但其实，接下来两个方法更直接一点

solution 2:

iterative
重点是维系3个节点prev, cur, next，这样才能完成链表的翻转

solution 3:

recursive
用递归的方式翻转，看起来只要维护两个节点，所以需要判断更多条件
