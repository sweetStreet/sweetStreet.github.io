[Leetcode92]Reverse_Linked_List_II

Yu' Blog
2019-10-09
algorithm, leetcode, reverse list

Reverse a linked list from position m to n. Do it in one-pass.

Note: 1 ≤ m ≤ n ≤ length of list.

Example:

Input: 1->2->3->4->5->NULL, m = 2, n = 4
Output: 1->4->3->2->5->NULL

solution

  1. dummy - track head position (dummy.next = head) case: if head is reversed
  2. pre - point to the start of the reversed list (0 to m-1)
  3. cur - point to beginning of sub-list to be reversed(itself val unchanged) (0 to n-m swap)
  4. nex - point to the node will be reversed(overall traverse the list)

这里之前nex.next = cur出问题是因为pre.next其实和cur是不一样的,cur只是next的前一个节点，而不是pre的后一个节点，画个图就明白了！！！

solution 2
设置start和tailnode直接让后面指向前面，这个挺好想明白的，注意换的次数
