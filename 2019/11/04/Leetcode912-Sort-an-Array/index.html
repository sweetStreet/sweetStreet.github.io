[Leetcode912]Sort_an_Array

Yu' Blog
2019-11-04
algorithm, leetcode, sort

quick sort
递归方法都有exit condition，这里就是要判断if(start>=end) return;
pivot(中轴元素) nums[start]
元素交换
递归条件 不要把left包含进去了
quicksort(nums, start, left-1);
quicksort(nums, left+1, end);

merge sort
split the array into two parts
sort two parts separately
merge two arrays
注意merge时, nums的起始索引是start
