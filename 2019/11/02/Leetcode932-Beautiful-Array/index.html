[Leetcode932]Beautiful_Array

Yu' Blog
2019-11-02
algorithm, divide and conquer, leetcode

For some fixed N, an array A is beautiful if it is a permutation of the integers 1, 2, …, N, such that:

For every i < j, there is no k with i < k < j such that A[k] * 2 = A[i] + A[j].

Given N, return any beautiful array A. (It is guaranteed that one exists.)

Example 1:

Input: 4
Output: [2,1,4,3]

Example 2:

Input: 5
Output: [3,1,2,5,4]

Note:
1 <= N <= 1000

solution
对于每一个数A[k]来说，任意i<k, j>k，需要满足A[i] + A[j] != 2A[k]，因为2A[k]一定是偶数，所以只需要A[i]和A[j]一个是奇数，一个是偶数。这样就可以很自然得想到divide and conquer。长度为N的序列中，奇数个数为（N+1）/2，偶数个数为N/2。那么只要在分隔符处，分隔符左边是奇数，分隔符右边是偶数，这样递归，就可以得到结果。
