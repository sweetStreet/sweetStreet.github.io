[Leetcode50]Pow(x,n)

Yu' Blog
2019-11-12
algorithm, leetcode

Description

Implement pow(x, n), which calculates x raised to the power n (xn).

Example 1:

Input: 2.00000, 10
Output: 1024.00000

Example 2:

Input: 2.10000, 3
Output: 9.26100

Example 3:

Input: 2.00000, -2
Output: 0.25000
Explanation: 2-2 = 1/22 = 1/4 = 0.25

Note:

-100.0 < x < 100.0
n is a 32-bit signed integer, within the range [−231, 231 − 1]

Solution

这是一道非常适合follow up的题
如果只是单纯得 return x*myPow(x, n-1) 时间复杂度是O(n)，会超时
因此如果递归的话，每次将n除2，但这需要我们将n的正负分开讨论，因为负数以为这x要取倒数
如果n是负数的话，又会遇到需要n等于Integer.MIN_VALUE的情况，在这样的情况下，-n是会越界的
还有就是0的0次方是多少，我们可以认为所有数的0次方都是1
讨论清楚所有的edge case之后就可以开始写题了
