[Leetcode1153]String_Transforms_Into_Another_String

Yu' Blog
2019-11-26
algorithm

Description

Given two strings str1 and str2 of the same length, determine whether you can transform str1 into str2 by doing zero or more conversions.

In one conversion you can convert all occurrences of one character in str1 to any other lowercase English character.

Return true if and only if you can transform str1 into str2.

Example 1:

Input: str1 = “aabcc”, str2 = “ccdee”
Output: true
Explanation: Convert ‘c’ to ‘e’ then ‘b’ to ‘d’ then ‘a’ to ‘c’. Note that the order of conversions matter.
Example 2:

Input: str1 = “leetcode”, str2 = “codeleet”
Output: false
Explanation: There is no way to transform str1 to str2.

Note:

1 <= str1.length == str2.length <= 10^4
Both str1 and str2 contain only lowercase English letters.

Solution

将字符转换用图表示 str1 -> str2 中字符 c1 -> c2
不能成功转换的情况

  1. 一个字符出度大于1
  2. 26个字符构成一个大环以致于找不到一个中间字符
    对于形成环的图，abc->bca，这个时候只要找到一个中间转换节点k，满足a->k->b就可以消除环
    例如 abc -> kbc -> kba -> kca -> bca

注意这里有个edge case, str1是不是和str2完全相同，如果相同可以提前返回true

special case:
“abcdefghijklmnopqrstuvwxyz”
“bcdefghijklmnopqrstuvwxyzq”
通过这个例子我们可以看出需要用到indegree来判断能不能找到中间字符

“abcdefghijklmnopqrstuvwxyz”
“abcdefghijklmnopqrstuvwxyz”
