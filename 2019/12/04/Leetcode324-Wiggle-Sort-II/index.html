[Leetcode324]Wiggle_Sort_II

Yu' Blog
2019-12-04
algorithm, leetcode

Description

Given an unsorted array nums, reorder it such that nums[0] < nums[1] > nums[2] < nums[3]….

Example 1:

Input: nums = [1, 5, 1, 1, 6, 4]
Output: One possible answer is [1, 4, 1, 5, 1, 6].
Example 2:

Input: nums = [1, 3, 2, 2, 3, 1]
Output: One possible answer is [2, 3, 1, 3, 1, 2].
Note:
You may assume all input has valid answer.

Follow Up:
Can you do it in O(n) time and/or in-place with O(1) extra space?

Solution

一个排好序的数组下标可以重新组合成下图
4 5 6 7
////
0 1 2 3
考虑中位数这个概念
特殊情况是中位数相等，在上图这个例子中要求，index为2 3 4可以相同，但如果超过3个中位数相同就不满足条件了
7 6 5 4
////
3 2 1 0
考虑重新组合，在遇到特殊情况，则可以允许index为2 3 4 5的数相同

但是题目的follow up要求时间复杂度为O(n)
则考虑
counting sort, radix sort, bucket sort

问题转化为
小于mid mid 大于mid 三个部分 (3 way partition)
可以考虑quick select
6 5 4
3 2 1 0
