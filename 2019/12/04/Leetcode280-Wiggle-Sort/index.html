[Leetcode280]Wiggle_Sort

Yu' Blog
2019-12-04
algorithm, leetcode

Description

New
Given an unsorted array nums, reorder it in-place such that nums[0] <= nums[1] >= nums[2] <= nums[3]….

Example:

Input: nums = [3,5,2,1,6,4]
Output: One possible answer is [3,5,1,6,2,4]

Solution

Solution1

排序，时间复杂度O(nlogn)

class Solution {
//1 3 2 5 4 6
public void wiggleSort(int[] nums) {
for(int i = 0; i<nums.length-1; i++){
if(i%2 == 0){
if(nums[i]>nums[i+1]){
swap(nums, i, i+1);
}
}else{
if(nums[i]<nums[i+1]){
swap(nums, i, i+1);
}
}
}
}

public void swap(int[] nums, int i, int j){
    int temp = nums[i];
    nums[i] = nums[j];
    nums[j] = temp;
}

}
